TCN-AFPpred

Accelerating the discovery of antifungal peptides using deep temporal convolutional networks

The increased application of machine intelligence in biological sciences has led to the development of several automated tools and techniques, thus enabling rapid drug discovery. TCN-AFPpred is one such attempt to predict and identify antifungal peptides in protein sequences of various organisms using Temporal Convolutional Networks (TCN) in order to accelerate the process of drug discovery against fungi that cause damage to plant, animal, and human health.

How does this tool work?


Predict AFPs in proteins Classify Peptides

The user has to supply a single protein sequence to run TCN-AFPpred. The sequences must be supplied in string format. Each sequence must contain at least four amino acids. Moreover, each sequence should contain only standard amino acid residues given in a single letter format, i.e., A, C, D, E, F, G, H, I, K, L, M, N, P, Q, R, S, T, V, W, Y. The output is provided as a table in an excel file. As soon as the output is ready, the user will be prompted to save the file in their system.

The user can provide an input comprising multiple peptide sequences in FASTA format. Each sequence must contain at least four amino acids. Moreover, each sequence should contain only standard amino acid residues given in a single letter format, i.e., A, C, D, E, F, G, H, I, K, L, M, N, P, Q, R, S, T, V, W, Y. The output is provided as a table in an excel file. As soon as the output is ready, the user will be prompted to save the file in their system.

The TCN-AFPpred model

The model behind this app is based on Temporal Convolutional Network (TCN). It uses the amino acid sequences of a given peptide as input and predicts whether the peptide is antifungal or not. The model consists of a word embedding layer, a spatial dropout layer, two TCN blocks, a global average pooling layer, a dense layer, and a dropout layer.
